# README

Sitio web estático de Postek USA.

## Estructura

- `index.html`: página principal
- `products.html`: productos
- `services.html`: servicios
- `about.html`: acerca de
- `contact.html`: contacto
- `styles/`: estilos
- `scripts/`: scripts
- `images/`: recursos visuales
- `docs/product_requirements.md`: documento de requerimientos de producto

## Cómo usar

1. Abrir `index.html` en el navegador.
2. Editar archivos HTML/CSS/JS según necesidad.

## Notas

Proyecto estático sin dependencias de build.
